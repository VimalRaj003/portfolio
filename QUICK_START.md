## 💾 Essential Operations

### 🔄 Backup Your Data
1. Click "Settings" in sidebar
2. Click "Export All Data"
3. JSON file downloads
4. Save to cloud/external drive
5. **Done!** You have a backup

### 📥 Restore Data
1. Click "Settings"
2. Click "Import Data"
3. Select your backup JSON
4. Click "Import"
5. **Done!** Your data is restored

## 📸 Image Upload Tips

### Recommended Sizes
- Profile Image: 400 × 400 pixels
- About Image: 500 × 500 pixels
- Project Images: 400 × 300 pixels
- Testimonial Avatars: 50 × 50 pixels

### File Types
✅ JPEG, PNG, WebP, GIF

### File Size
- Keep each image under 200 KB
- Total storage limit: ~5-10 MB

### Best Practices
- Use professional photos
- Compress before uploading
- Use consistent style
- High quality images

## 🎮 Common Tasks

### Add a New Project
1. Click "Manage Projects"
2. Click "Add New Project"
3. Fill in: Name, Description, Category
4. Upload project image
5. Add technologies (comma separated)
6. Add demo & GitHub URLs
7. Click "Save Project"
8. ✅ Done! View on main website

### Add a New Skill
1. Click "Manage Skills"
2. Click "Add New Skill"
3. Enter: Category, Name
4. Set proficiency level (0-100%)
5. Click "Save Skill"
6. ✅ Done!

### Add a Testimonial
1. Click "Manage Testimonials"
2. Click "Add New Testimonial"
3. Enter: Client name, title, testimonial text
4. Upload client photo
5. Set star rating
6. Click "Save Testimonial"
7. ✅ Done!

### Update Hero Section
1. Click "Edit Content"
2. Click "Hero Section" tab
3. Update: Title, subtitle, description
4. Upload new profile image
5. Click "Save Changes"
6. ✅ Instantly appears on website!

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't login | Check username/password (admin/password123) |
| Changes not showing | Refresh the main website |
| Images not uploading | Check file size (< 200 KB) |
